# Day 1 - LLM Fundamentals

## What is an LLM?
Ans: Large Language Model is basically a system  which is trained on large sets of data from internet in the initial stage of pretraining  to predict the next token based on previous tokens.

## What is Next Token Prediction?
Ans: it is the process of predicting the most likely next token based on the previous token by a LLM.

## What is a Token?
Ans: It is a basic unit of text which can be a word/part of word/punctuation processed by the LLM.

## What is a Context Window?
Ans: It is the maximum amount of text(in tokens) that a LLM can consider at a time of generating a response.

## What is Temperature?
Ans: It controls the randomness of the LLM's output, lower temperature is more predictable while higher temperature is more creative.
## Why do Hallucinations happen?
Ans: Hallucinations happens becuase LLM generates texts based on learned probabilities rather than verifying facts therefore sometimes the response generated by the LLM maybe incorrect.
